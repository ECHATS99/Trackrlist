# Trackrlist
Tracklist Browser Tool est un outil léger et pratique conçu pour Kali Linux. Il automatise les recherches OSINT en ouvrant automatiquement des fenêtres de navigateur avec des liens ciblés basés sur un nom, prénom, numéro ou autre indice fourni. L’outil supporte Chromium ou Firefox et nécessite uniquement leurs dépendances pour fonctionner. 

Dépendance 
Tracklist depends Des commandes

sudo apt update
sudo apt install chromium firefox curl -y
git clone https://github.com/ECHATS99/Trackrlist.git 
chmod +x trackrlist.sh
./trackrlist.sh
